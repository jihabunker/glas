# How to install python3 on Linux
<pre><code>
 ~]# wget https://www.python.org/ftp/python/3.6.3/Python-3.6.3.tar.xz
 ~]# xz -d Python-3.6.3.tar.xz
 ~]# tar -xvf Python-3.6.3
 ~]# cd Python-3.6.3
 ~]# ./configure
 ~]# make
 ~]# make install
</code></pre>
